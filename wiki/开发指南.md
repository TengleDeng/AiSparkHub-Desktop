# 开发指南

## 环境准备
- Python 3.9 及以上
- 建议使用虚拟环境
- 安装依赖：`pip install -r requirements.txt`

## 目录结构
- main.py：主入口
- app/components：界面组件
- app/controllers：控制器
- app/models：数据与数据库
- app/utils：工具函数

## 主要模块说明
- MainWindow：主窗口
- AuxiliaryWindow：辅助窗口
- WindowManager：窗口管理
- ThemeManager：主题管理
- DatabaseManager：数据库

## 贡献方式
1. Fork 本仓库
2. 新建分支开发
3. 提交 PR
